# Welcome!

Welcome to GuardPy. So, here’s the deal: this project is all about online connectivity, database syncing, and a chat system that actually works. I started this whole thing purely for the sake of enthusiasm—just wanted to build something cool, see how it handles live data, and have a solid tool for my own projects. 

Honestly, it’s a bit of a Frankenstein’s monster right now because I’ve been throwing everything at it to see what sticks. It’s written in a mix of Java, Zig, JavaScript, Go, C++, and C, plus there’s some Git and JSON stuff holding the configurations together. It’s definitely a work in progress, but it’s fast and it does exactly what I needed it to do. If you’re into low-level systems or just curious about how this stack comes together, feel free to poke around.
